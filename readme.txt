1. To run the app.py, you will need to activate virtual environment first and install all required packages (flask, pandas, math, graphviz and sklearn).
2. In the terminal in venv, type "flask run". 
3. Click on http://127.0.0.1:5000 and you will be redirected to the web page where the user can input its preferences.
4. After clicking on "Submit" button, the user will be redirected to another page where the prediction will be displayed along with the decision tree. 
5. The entropy of the last column is displayed in the terminal as requested by TA.
